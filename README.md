# Filipa Rebelo · Nutricionista

Website da Filipa Rebelo, Nutricionista (C.P. 2767N) com consultas em Viseu, Seia e Online.

## Stack

- React 18
- CSS custom properties (design tokens)
- Google Fonts (Cormorant Garamond + DM Sans)

## Instalação

```bash
npm install
npm start
```

## Build para produção

```bash
npm run build
```

## Deploy (GitHub Pages)

```bash
npm install --save-dev gh-pages
# Adicionar ao package.json: "homepage": "https://renasfp.github.io/filipa-rebelo-nutricao"
npm run build
npx gh-pages -d build
```
